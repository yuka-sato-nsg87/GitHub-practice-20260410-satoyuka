# Git / GitHub 入門講習

Git および GitHub の基本的な使い方を、実際に手を動かしながら学ぶための教材です。

## 対象

- 研究室新規配属の学生

- Git / GitHub の基礎について復習したい方

- その他で興味のある方

## 内容

学習者の方は、教材本体[`docs/git_github_beginner_textbook.md`](/docs/git_github_beginner_textbook.md) を参照してください。

| 章・補足 | 内容 |
|---|---|
| 1章 | Git / GitHub とは何か |
| 2章 | Git が管理するもの（リポジトリ、ファイル状態、.gitignore） |
| 3章 | 基本操作（add / commit / push / pull） |
| 4章 | 個人運用の標準フロー（branch / pull request / merge） |
| 5章 | 研究・論文執筆に向けた運用 |
| 補足 | よくあるミス、発展操作、fork、権限、LICENSE、README、ディレクトリ構成、Zip化されたリポジトリのGitHubリポ作成方法 |
